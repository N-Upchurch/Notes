# `for` Loops

``` python
# Example 1
print('Hi, my name is')
for i in range(2):
    print('chicka ' + str(i))
print('slim shady')

# Example 2
total = 0
for num in range(100):
    total = total + num
print (total)

# Example 3, setting a lower and upper range, with incrementor 
count = 0
for num in range (50, 505, 5):
    print (num)
```
